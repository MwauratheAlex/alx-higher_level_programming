=============
1-my_list.py
=============
Module contains a class ``MyList``
The class inherits from the list class
The class contains a function ``print_sorted(self)``
that prints a sorted list in ascending order

======
Usage:
====== 
First import:
	>>> MyList = __import__('1-my_list').MyList

Basic usage:
	>>> my_list = MyList()

	>>> my_list.append(1)

	>>> my_list.append(5)

	>>> my_list.append(3)

	>>> my_list.append(4)

	>>> print(my_list)
	[1, 5, 3, 4]

	>>> my_list.print_sorted()
	[1, 3, 4, 5]

Negative number in list:
	>>> my_list.append(-4)

	>>> print(my_list)
	[1, 5, 3, 4, -4]

	>>> my_list.print_sorted()
	[-4, 1, 3, 4, 5]

Repeated negative numbers:
	>>> my_list.append(-4)

	>>> my_list.print_sorted()
	[-4, -4, 1, 3, 4, 5]

Repeated positive numbers:
	>>> my_list.append(2)

	>>> my_list.append(2)

	>>> my_list.print_sorted()
	[-4, -4, 1, 2, 2, 3, 4, 5]
